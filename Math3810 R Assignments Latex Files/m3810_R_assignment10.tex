\section*{Instructions}
Show all reasoning clearly. All simulation results should be reproducible and clearly labeled.
You may use R for all computations.

\section*{Problems}
\begin{enumerate}[leftmargin=*]

\item \textbf{Covariance and Correlation}
\begin{enumerate}
\item Simulate 5000 pairs $(X,Y)$ with $X \sim N(0,1)$, $Y \sim N(0,1)$ independent.
\item Compute sample covariance and correlation.
\item Create $Z = X+Y$ and compute mean, variance, and correlation with $X$.
\end{enumerate}

\item \textbf{Dependent Variables}
\begin{enumerate}
\item Simulate $Y = 0.5 X + \epsilon$ with $\epsilon \sim N(0,1)$.
\item Compute sample covariance and correlation between $X$ and $Y$.
\item Plot scatterplot.
\end{enumerate}

\item \textbf{Linear Combinations}
\begin{enumerate}
\item Compute $W = 2X - 3Y$ and its sample mean and variance.
\item Compare with theoretical values using variance formulas.
\end{enumerate}

\item \textbf{Discussion}
\begin{itemize}
\item Explain how covariance and correlation describe linear dependence.
\item Discuss how correlation affects variance of sums and differences.
\end{itemize}

\end{enumerate}

\ifnum\printsol=1
\section*{Solutions}
\begin{lstlisting}[language=R]
set.seed(123)
X <- rnorm(5000)
Y <- rnorm(5000)
cov(X,Y); cor(X,Y)
Z <- X+Y
mean(Z); var(Z); cor(Z,X)

epsilon <- rnorm(5000)
Y2 <- 0.5*X + epsilon
cov(X,Y2); cor(X,Y2)
plot(X,Y2)

W <- 2*X - 3*Y2
mean(W); var(W)
\end{lstlisting}

Correlation measures linear dependence. Covariance sign indicates direction.
Variance of sums increases with positive correlation; variance of differences decreases.
